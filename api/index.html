OneTwoTrie API

Comment utiliser l'API ?

https://vexcited.github.io/onetwotrie/api/product.html

Paramètres
?q={query}
Recherche dans `code-barres` ; `nom` ; `marque`

?cb={query}
Recherche un code-barre

Exemples

(...)/product.html?cb=3274080005003
