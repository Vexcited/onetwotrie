API

Pour les développeurs


Endpoint API

https://www.vexcited.ml/api/onetwotrie


GET /searchProduct

Recherche un produit en fonction du paramètre q.
Peut être le nom du produit, sa marque ou son code-barres.

q - Query (Obligatoire)


Exemple de retour avec /searchProduct?q=Cristaline

{
 "status": true,
 "query": "Cristaline",
 "num_rows": 1,
 "results": [
  {
   "productID": 1,
   "barcode": "3274080005003",
   "name": "Eau de source 150cl",
   "brand": "Cristaline",
   "type": "bouteilleplastique",
   "image": "https://raw.githubusercontent.com/Vexcited/onetwotrie/master/src/api/images/3274080005003.jpg",
   "alimentaire": "no"
  }
 ]
}


GET /searchCity

Recherche une ville en fonction du paramètre q.
Peut être le nom de la ville ou du code-postal.

q - Query (Obligatoire)


Exemple de retour avec /searchCity?q=Limoges
